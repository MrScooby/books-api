Simple API for private use.

## Description

API for simple app displaying data about books that I reed.
Web app repo also available on my git profile.

## Technologies (for now)

- Typescript
- Nest
- Swagger
- postgres
- cheerio

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
